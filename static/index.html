Quick Start Guide

Fun Videos — Start here. Drop a photo, let AI generate a creative video with matching music. Uses local Ollama — no API keys needed.
↓ generated clips flow automatically
Video Bridges — Got 2+ clips? Drop them here and AI will generate smooth cinematic transitions between them. Clips from Fun Videos appear automatically.
↓ or start with images
SD Prompts — Drop a reference image, get Stable Diffusion prompts in three regional columns (Left / Center / Right) for Forge Couple. Generate the image directly in Forge, then send it to Fun Videos.
↓ Tools dropdown for utilities
Video Tools — Reverse, mirror, speed up/slow down, upscale, or convert any video. Drop files or paste paths.
Image to Video — Turn a folder of photos into a Ken Burns slideshow with zoom and crossfade effects. No AI needed.

Tips

  • You can use any single tab standalone — you don't have to follow the full pipeline.
  • Every generated file is tracked in the session and appears as an option in other tabs.
  • In SD Prompts, type __wildcard__ or {option1|option2} and Forge will expand it randomly on each generation.
  • Services (WanGP, ACE-Step, Forge) only need to be running for features that use them.
